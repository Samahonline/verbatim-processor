# Verbatim Data Processor

A Streamlit app for automatically processing verbatim text responses from survey data.

## Features
- Automatically detects verbatim columns
- Excludes NULL, date, and address columns
- Creates individual Excel sheets for each question
- Downloads formatted output workbook
